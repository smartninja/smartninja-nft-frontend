# SmartNinja NFTs

## Quickstart

```bash
npm install
npm run dev
```

## Deploy

npm run build && cd dist
git init && git add -A
git commit -m "message"
git push -f git@github.com:smartninja/smartninja-nft-frontend.git main:gh-pages
